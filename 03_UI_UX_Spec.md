# UI / UX Specification

## Design Philosophy
Modern enterprise SaaS inspired by Linear, Stripe and Notion.

## Layout
Three-column layout.

### Header
- ProcureIQ logo
- Workspace Toggle (Manager / Head)
- Sourcing Event Dropdown
- Export PDF
- Export Excel

### Left Panel
- KPI Cards
- Suggested Questions

### Center
- Supplier Comparison Table
- Sticky header
- Horizontal scroll

### Right
- AI Recommendation Card
- Why this supplier?
- Trade-offs
- Chat
- Procurement Summary
